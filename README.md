## 🚀 How to Use

### Step 1: Add Your Images
Create an `images` folder in your repository and add 6 photos:
- `meeting.jpg` - Your first meeting
- `conversation.jpg` - Getting to know each other
- `moment.jpg` - An unforgettable moment
- `realization.jpg` - A candid photo
- `question.jpg` - A photo for the big moment
- `future.jpg` - A dream photo for your future

### Step 2: Customize the Content
Open `index.html` and replace the placeholder text with your own personal messages:
- Change the section titles if you want
- Add your own heartfelt messages
- Personalize dates and moments

### Step 3: Update Image References
In `index.html`, replace the `<div class="image-placeholder">` sections with actual images:

```html
<img src="images/meeting.jpg" alt="Our First Meeting" class="timeline-image">
```

### Step 4: Deploy to GitHub Pages
1. Go to your repository settings
2. Scroll to "Pages" section
3. Select "main" branch as source
4. Save and wait for deployment
5. Your site will be live at: `https://4ktlony.github.io/smiley/`

## 🎨 Customization

### Change Colors
In `style.css`, find the gradient colors and change them:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

Popular color combinations:
- **Pink Theme**: `#ff6b6b` to `#ee5a6f`
- **Blue Theme**: `#4facfe` to `#00f2fe`
- **Orange Theme**: `#fa7e1e` to `#d62828`

### Add More Timeline Items
Copy this structure and paste it between timeline markers:
```html
<!-- Timeline Dot -->
<div class="timeline-marker"></div>

<!-- Timeline Item -->
<div class="timeline-item left/right">
    <div class="timeline-content">
        <h2>Your Title</h2>
        <p class="date">When</p>
        <p>Your message here</p>
        <div class="image-placeholder">📸 Your Photo</div>
    </div>
</div>
```
